# Catch the Fruit (과일 받기 게임)

## 1. 게임 개요
하늘에서 떨어지는 과일을 바구니로 받아 점수를 획득하는 게임입니다.
플레이어는 웹캠 앞에서 몸을 움직여 바구니를 조작합니다.

## 2. 조작 방법 (Pose)
Teachable Machine 모델의 3가지 클래스를 사용합니다:
- **LEFT (왼쪽)**: 몸을 왼쪽으로 기울임 -> 바구니가 왼쪽(Left Lane)으로 이동
- **CENTER (중앙)**: 정면을 바라봄 -> 바구니가 중앙(Center Lane)으로 이동
- **RIGHT (오른쪽)**: 몸을 오른쪽으로 기울임 -> 바구니가 오른쪽(Right Lane)으로 이동

*화면은 3개의 레인(Lane)으로 구성되며, 포즈에 따라 바구니가 해당 레인으로 이동합니다.*

## 3. 아이템 및 점수
| 아이템 | 이미지/설명 | 점수/효과 |
|---|---|---|
| 🍎 사과 | 빨간 사과 | +100점 |
| 🍊 오렌지 | 주황 오렌지 | +200점 |
| 💣 폭탄 | 검은 폭탄 | **게임 오버** (또는 점수 차감) |

## 4. 게임 규칙
- **제한 시간**: 60초 동안 가능한 많은 점수 획득
- **게임 오버 조건**: 
  1. 폭탄을 받았을 때
  2. 제한 시간이 종료되었을 때
- **레벨 업**: 
  - 점수 500점마다 레벨 증가
  - 레벨이 오르면 과일이 떨어지는 속도가 빨라짐

## 5. 화면 구성 (UI)
- **Play 화면**:
  - 상단: Score(점수), Time(남은 시간), Level(난이도) 표시
  - 배경: 하늘 또는 심플한 단색 배경
  - 캐릭터: 바구니 (또는 사람 캐릭터)
- **Game Over 화면**:
  - 최종 점수 표시
  - '다시 시작' 버튼
